Fonte Record Type (Marca Record / MKTID)

Copiar para esta pasta os ficheiros TTF da pasta da documentação:
  documentação\Assets\Assets\Font\RecordType\Desktop\TTF\

Ficheiros necessários:
  - RecordType-Regular.ttf
  - RecordType-Medium.ttf
  - RecordType-Bold.ttf

Sem estes ficheiros, a aplicação usa a fonte de fallback (Geist).
